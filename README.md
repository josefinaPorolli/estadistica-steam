# estadistica-steam

Generador de reportes de frecuencias para datos de juegos de Steam.

## Requisitos

- Python 3

## Instalación

```bash
chmod +x install.sh boot.sh
./install.sh
```

## Ejecución

```bash
./boot.sh
```

El reporte se genera en `reportes/frecuencias.md` y las imágenes en `images/`.
