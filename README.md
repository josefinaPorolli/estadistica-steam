# estadistica-steam

Generador de reportes de frecuencias para datos de juegos de Steam.

## Requisitos

- Python 3

## Instalación

```bash
chmod +x install.sh boot.sh
./install.sh
```

## Ejecución

```bash
./boot.sh
```

Los reportes se generan por variable en `reportes/variables/`, con un índice en `reportes/index.md`, y las imágenes en `images/`.
